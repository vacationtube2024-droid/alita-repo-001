# Auto-Code Documenter

An AI-powered tool that automatically generates documentation for GitHub repositories.

## Features

- [ ] Scan repository structure
- [ ] Analyze code files (Python, JavaScript, etc.)
- [ ] Generate README.md automatically
- [ ] Add docstrings to code
- [ ] Create API documentation

## Usage

```bash
python doc_generator.py --repo owner/repo --output ./docs
```

## Requirements

- Python 3.8+
- GitHub CLI (gh)
- OpenAI API key (optional for advanced features)
